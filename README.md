# Telegram-бот «Персональный трекер и аналитик сна» 

## 📌 Описание проекта
Данный проект представляет собой Telegram-бота для мониторинга биоритмов и анализа качества сна пользователя. В отличие от автоматических трекеров, система базируется на ретроспективном ручном вводе данных, что обеспечивает гибкость внесения записей за любой прошедший период и исключает сбои при потере интернет-соединения в момент засыпания.

Программа рассчитывает длительность отдыха, фиксирует субъективное самочувствие пользователя и визуализирует динамику режима в виде аналитических графиков.

##
* **Язык программирования:** Python 3.14+
* **Библиотека API:** `pyTelegramBotAPI` (для взаимодействия с серверами Telegram)
* **Система управления базами данных (СУБД):** `SQLite3` (встраиваемая, реляционная БД)
* **Визуализация данных:** `Matplotlib` (для динамической генерации графиков)
* **Работа с временными метками:** `datetime`

## Основной функционал и архитектура
1. **Модуль сбора данных (UX/UI):** Пошаговая анкета ввода (`bot.register_next_step_handler`) с полной валидацией формата времени по маске `ДД.ММ.ГГГГ ХХ:ММ`. Присутствует защита от логических ошибок (например, ввод времени пробуждения раньше времени засыпания).
2. **Модуль хранения (Data Layer):** Локальная база данных `sleep_tracker.db`. Структура таблицы включает поля для идентификатора пользователя, меток времени, рассчитанной длительности и оценки самочувствия.
3. **Модуль аналитики (BI Component):** Извлечение последних 7 сессий сна из БД, агрегация данных и построение линейного графика с отображением «линии нормы» (8 часов) для выявления дефицита сна.

## Инструкция по локальному развертыванию

1. Клонируйте репозиторий:
   ```bash
   git clone [https://github.com/ВАШ_НИК/НАЗВАНИЕ_РЕПО.git](https://github.com/ВАШ_НИК/НАЗВАНИЕ_РЕПО.git)

2. Установите необходимые зависимости:
    Bash
    python -m pip install -r requirements.txt

3. Создайте файл конфигурации config.py в корневой папке проекта и укажите ваш API-токен:

    Python
    TOKEN = "ВАШ_ТЕЛЕГРАМ_ТОКЕН"

4. Запустите главный скрипт:

    Bash
    python main.py
